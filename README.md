# 프로젝트명 (추후 업데이트)

- 올해의 게임 리스트에 사용자의 평점과 리뷰를 남기고, 그들의 커뮤니티를 형성할 수 있는 웹 사이트 제작.

## teamRules

- 파일 네이밍의 경우, 붙어있는 단어는 camelCase 이용해 주시면 감사하겠습니다.
- 변수 네이밍 진행 시, snakeCase 이용하여 상위개념과 하위개념을 설명해 주시면 감사하겠습니다.
  ex)joinPage_pwTest
